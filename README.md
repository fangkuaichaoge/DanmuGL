# 🎮 DanmuGL | AI Live Barrage Mod for Minecraft Mobile
Ever feel lonely playing Minecraft alone?  
This mod generates **real-time AI barrage comments** based on your gameplay screen, bringing you the authentic streamer audience atmosphere!

Barrage (danmu) is a popular culture on Chinese and Japanese anime & livestream platforms, now available for your single-player Minecraft.

## 📥 Download Links
**DanmuGL Mod**  
https://github.com/fangkuaichaoge/DanmuGL

**Required LeviLauncher**  
https://github.com/LiteLDev/LeviLaunchroid

## 📝 Setup Tutorial
1. Download the `.so` mod file
2. Import and enable it in LeviLauncher
3. An ImGui configuration panel will appear in-game
4. Edit your config file path:  
`/storage/emulated/0/games/DanmuGL/config.json`
5. Fill in your API Key, Base URL and model parameters

## ✅ Free Recommended AI API
### Global Users (100% Free)
**Agnes AI**
Website: https://agnes-ai.com/
Base URL: `https://apihub.agnes-ai.com/v1/chat/completions`
Model: `agnes-1.5-flash`

### China Users (Free Vision Model)
**Zhipu Qingyan（智谱清言）**
Website: https://bigmodel.cn/
Base URL: `https://open.bigmodel.cn/api/paas/v4/chat/completions`
Model: `GLM-4V-Flash`

## ⚙️ Adjustable Options
Tweakable in ImGui menu:
- AI temperature (output creativity)
- AI character preset & tone

## 🌐 Language Support
- Only **English & Chinese** are supported
- Chinese display requires a valid `.ttf` font path filled in the config file

## ⚠️ Important Notes
- Free platforms like Zhipu and SiliconFlow work stably
- Individual third-party APIs may have compatibility issues
- AI barrages inevitably feel robotic compared to real human comments, prompt optimization is ongoing
- English barrage quality is lower due to the lack of barrage culture in Western datasets
